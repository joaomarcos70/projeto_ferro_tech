## Como executar aplicação:

    * git clone no repositório
    * Acessar na pasta da aplicação
    * Acessar a pasta banco de dados
    * Executar script (MySQL)
    * Abrir o CMD na pasta
    	 
    $ npm install
    $ npm run start
